About this project

Aesthetics-KG is a knowledge graph of 1,031 internet aesthetics, capturing motifs, vibes, cultural context, and connections between aesthetic communities. It was built from the Aesthetics Wiki using RML mappings and is served as Linked Open Data via a SPARQL endpoint.

The project was presented at ISWC 2026 as:
Vibes Lore Core: the Aesthetics Knowledge Graph
